# Personal Expense Tracker

Simple console-based Personal Expense Tracker in Python.

## Features
- Add, view, edit, delete expenses
- Group by category
- Show total and category summaries
- Filter by month/year
- Visualize summary with matplotlib (pie chart)
- CSV-based storage (`data/expenses.csv`)
